# EduNotes Hub

📚 **All Your Notes, All Exams, One Place!**

EduNotes Hub is a modern, responsive educational notes platform where students can explore and download notes for:
- Graduation (Engineering, Commerce, Arts, Science)
- Post-Graduation (MBA, M.Tech, M.Sc, MA)
- JEE Preparation (Physics, Chemistry, Mathematics)
- CUET (General Test, Domain Specific, Language)
- Class 10th (Math, Science, English, Social Studies)
- Class 12th (PCM, PCB, Commerce, Humanities)

---

## ✨ Features
- 📂 Browse notes by categories (Graduation, JEE, CUET, etc.)
- 📑 Featured notes with preview & download
- ⭐ Student testimonials
- 📧 Newsletter subscription
- 📞 Contact & support section
- 💳 Payment support (UPI, GPay, Paytm)
